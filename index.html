    Welcome to my game's dev site!

    I'm making a game in C# using the XNA framework. It's a 2D platformer. Right now I'm focusing on coding and getting something to work, rather than figuring out the premises of my game. I'm thinking Hitchiker's Guide theme, with Arthur Dent as the main character. Other than that, my game has almost no direction. Should be fun.

    Here is where all the action on my game's development happens. You can follow changes as they happen by clicking View on Github above. You'll be able to see what I've done so far, the source code, and what still needs to be done.

    Have an idea for this game? Questions about it?

    Please email me! I'd love to hear from you.

    Want to help?

    Please do! Email me and I'll help you get set up and put you in the loop. If you're especially brave, you can go right ahead and checkout the git repository and start coding. (or drawing or writing or whatever)

    Want to try the game out?

    Right now the game is very much in development. At time of writing, I'm using graphics I stole from a friend's game, there is one platform, the character cannot walk or jump, and collision detection is very broken. Also, there are no installers for the game, so you'll have to build it yourself. If that hasn't convinced you that my game sucks yet, you can download the source code and prove it to yourself. Visual studio and XNA required to build.
